# Student Performance Management System

A Python-based project to manage student records such as marks, attendance, grades, and pass/fail status.

## Features

* Add Student
* View Students
* Search Student
* Update Marks
* Delete Student
* Find Topper
* Average Marks
* CSV File Storage

## Technologies Used

Python, CSV File Handling, List, Dictionary, Loops, Functions

## Learning Outcome

Improved Python logic building, CRUD operations, and data handling skills.
